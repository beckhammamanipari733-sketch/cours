EST316

Arquitectura de Redes y Protocolos

3.0 créditosRedes

Sumilla

Diseño y configuración de topologías de red, direccionamiento IP y servicios básicos (DNS, DHCP, HTTP), con práctica intensiva en Cisco Packet Tracer.

Objetivos

  • Diseñar topologías LAN/WAN con direccionamiento IP coherente.
  • Configurar routers y switches mediante CLI.
  • Implementar servicios de DHCP, DNS y Web en la red simulada.
  • Verificar conectividad y diagnosticar fallas de enrutamiento.

Unidades temáticas

  • Modelo OSI/TCP-IP y direccionamiento IP
  • Configuración de routers y switches
  • Servicios de red: DHCP, DNS, HTTP
  • Topologías multi-router y verificación end-to-end
